# npm Packaging Design — @chocks/locode

Date: 2026-03-04

## Goal

Publish locode as `@chocks/locode` to npm as a public, MIT-licensed package that users can install globally with `npm install -g @chocks/locode`.

## Decisions

- **Package name**: `@chocks/locode` (scoped, public)
- **License**: MIT
- **Distribution**: pre-built `dist/` only — no TypeScript source shipped
- **Config**: `locode.yaml` not bundled; generated by `locode setup` on first run
- **Publish method**: manual (`npm publish`) for now

## Changes Required

### `package.json`
- Set `name` to `@chocks/locode`
- Set `version` to `0.1.0`
- Add `"license": "MIT"`
- Add `"files": ["dist"]`
- Add `"engines": { "node": ">=18" }`
- Remove `"main": "index.js"` (not a library)
- Add `"prepublishOnly": "npm test && npm run build"`

### `.npmignore`
Exclude: `src/`, `benchmark/`, `node_modules/`, `*.ts`, `tsconfig.json`, `locode.yaml`, `vitest.config.ts`, `.locode/`

### `LICENSE`
Add MIT license file.

### `dist/src/index.js`
Ensure shebang (`#!/usr/bin/env node`) is present and file is executable (`chmod +x`).

### `README.md`
- Update install command to `npm install -g @chocks/locode`
- Add TODO for GitHub Actions automated publish

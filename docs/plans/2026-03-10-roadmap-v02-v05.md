# Locode Roadmap — v0.2 → v0.5

**Date:** 2026-03-10
**Status:** Proposed

---

## Version Progression

Each version builds on the previous. No version can be skipped.

```
v0.1 (current)          v0.2                    v0.3                    v0.4                    v0.5
─────────────────────── ─────────────────────── ─────────────────────── ─────────────────────── ───────────────────────
Routing CLI             Tool Foundation         Coding Agent            Codebase Intelligence   Workflow Automation
                        │                       │                       │                       │
• Regex + LLM routing   │ • Tool registry       │ • Agent loop          │ • File tree index     │ • Workflow engine
• LocalAgent (Ollama)   │ • Tool executor       │   (plan→edit→validate)│ • Symbol index        │ • Milestone commands
• ClaudeAgent           │ • Safety gate         │ • Code editor         │   (tree-sitter WASM)  │ • State persistence
• Token tracking        │ • Migrated tools      │ • Diff renderer       │ • Embedding index     │ • Context curation
• REPL + single-shot    │ • search_code tool    │ • Agent memory        │   (Ollama vectors)    │ • Context cache
• MCP integration       │ • write_file tool     │ • Streaming UX        │ • Dependency graph    │ • Claude structured I/O
                        │ • Schema validation   │ • Edit planning       │ • Context retriever   │ • Error recovery
                        │                       │ • Model routing/phase │ • Budget manager      │ • Progress display
                        │                       │                       │ • Incremental updates  │
```

---

## Dependency Chain

```
v0.2 Tool Foundation
 └──► v0.3 Coding Agent (uses ToolExecutor from v0.2)
       └──► v0.4 Codebase Intelligence (plugs into Agent's ANALYZE phase)
             └──► v0.5 Workflow Automation (chains Agent runs + uses Index for context curation)
```

## Config Evolution

Each version adds config sections. All are optional with sensible defaults.

| Version | New Config Sections |
|---|---|
| v0.2 | `safety` (always_confirm, auto_approve, allowed_write_paths) |
| v0.3 | `agent` (max_iterations, auto_confirm, validation_command), `editor` (show_diff, backup) |
| v0.4 | `index` (languages, embedding_model, ignore patterns), `context_retrieval` (max_files, token budget) |
| v0.5 | `workflows` (storage_dir, custom_templates_dir) |
